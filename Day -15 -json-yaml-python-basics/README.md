#### Json (JavaScript Object Notation) 
 - JSON is a lightweight, text-based format for storing and exchanging data. 
 - It is easy for humans to read and write, and easy for machines to parse and generate.
 ```
{
  "name": "Dinesh",
  "age": 21,
  "skills": ["Python", "Mysql"]
}
```
